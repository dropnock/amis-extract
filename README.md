# amis-extract

AMIS Extract is a utility to create an API to access data that is published weekly by the Ministry of Agriculture in Jamaica. The API allows users to access this data for further data analysis and to gain insight on the availability and prices of agriculture produce in Jamaica.

AMIS Extract is based on FastAPI and can be setup by following these steps:

### Setting up the environment
Step 1 - Clone the repository to your local machine.
Step 2 - Setup the python environment by importing the requirements

 > pip -r requirements.txt

### Starting the API
Step 3 - Start the FastAPI 

> uvicorn main:app


Documentation for the API can be found at [Documentation](http://127.0.0.1:8000/docs "http://127.0.0.1:8000/docs")
